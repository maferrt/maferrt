## 🚀 Proyectos destacados

### 📚 Chronicles of Elia

Aplicación para aprender inglés mediante rutas personalizadas de acuerdo con el nivel, perfil profesional, necesidades reales e intereses de cada usuario.

Este proyecto combina mi experiencia en enseñanza de idiomas con el desarrollo de software. La experiencia está inspirada en los videojuegos RPG: cada usuario avanza mediante misiones, evaluaciones, niveles y árboles de habilidades acompañado por **Elia**, el avatar guía de la plataforma.

#### Funcionalidades implementadas

- Registro e inicio de sesión conectados con el backend.
- Autenticación y autorización mediante JWT.
- Persistencia de usuarios en PostgreSQL.
- Validación de datos y manejo de errores.
- Detección de correos previamente registrados.
- Flujo inicial de onboarding.
- Pantallas de registro e inicio de sesión.
- Primeros endpoints para la evaluación diagnóstica.
- Comunicación entre frontend y API REST.
- Base de datos ejecutada mediante Docker.
- Arquitectura preparada para continuar creciendo de forma modular.

#### En desarrollo

- Evaluación diagnóstica completa.
- Creación de perfiles de aprendizaje.
- Rutas personalizadas por nivel y profesión.
- Seguimiento de progreso.
- Sistema de misiones y habilidades.
- Implementación progresiva de microservicios.
- Integración de funcionalidades asistidas por inteligencia artificial.

**Tecnologías:**  
`Java` · `Spring Boot` · `Spring Security` · `JWT` · `PostgreSQL` · `Docker` · `TypeScript` · `React`

**Estado:** En desarrollo activo.

### 🦉 Trinity | Psicoterapia Integral

Plataforma freelance desarrollada para una profesional de psicoterapia integral y biodescodificación.

El proyecto fue creado a partir de las necesidades reales de la clienta y contempla tanto la presentación de sus servicios como la integración de funcionalidades comerciales y de contacto.

La interfaz cuenta con una identidad visual cálida y responsiva, acompañada por **Trinity**, un búho que funciona como personaje representativo de la plataforma.

#### Funcionalidades implementadas

- Landing page informativa.
- Catálogo de servicios.
- Sección para conocer a la especialista.
- Testimonios y contenido informativo.
- Registro e inicio de sesión.
- Inicio de sesión con Google.
- Carrito de compras.
- Formulario de contacto.
- Envío de correos mediante Resend.
- Diseño responsivo para dispositivos móviles y escritorio.
- Integración con servicios externos.
- Despliegue y mantenimiento en producción.

#### Próximas implementaciones

- Chatbot representado por Trinity.
- Test orientativo asistido con inteligencia artificial.
- Plataforma para seguimiento de pacientes.
- Mejoras continuas de experiencia de usuario.

**Tecnologías:**  
`Next.js` · `TypeScript` · `React` · `Tailwind CSS` · `APIs REST` · `Resend` · `Vercel`

**Estado:** Publicado en producción y en mejora continua.
